<h3><b>Программа проверяет подтверждает ли число гипотезу Коллатца.</b></h3>
<hr>

<br><br>

<h1>Arguments:</h1>
<h2>Prefix: - or --</h2>
<h3><b>

* help : Print all arguments <br><br>
* search_mode= : set search_mode:<br>
  0 - default ; 1 - enumeration mode<br>
  template default: 10 12 - program checking numbers 10 and 12<br>
  template enumeration: 10 12 - program checking numbers from 10 to 12<br><br>
* cycles_limit= : updating cycles limit<br>

</h3></b>
